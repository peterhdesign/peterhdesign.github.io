Role

UX Design

Art Direction

Tools

Time to Watch is a website that highlights different watches in cinema. The website reveals the significance and impact that different watches have on their movies as well as information about the watch itself.

The website was created using HTML and CSS. The slideshow function is initialized using Swiper.
